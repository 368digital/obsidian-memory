# obsidian-memory

## Session Tracking (ОБЯЗАТЕЛЬНО)

При КАЖДОЙ сессии:

### Старт сессии (первое сообщение, БЕЗ ИСКЛЮЧЕНИЙ):
1. Проверь `claude-memory/.changed` — если есть, прочитай и обработай изменения из Obsidian
2. Прочитай `claude-memory/MEMORY.md` — загрузи контекст
3. Прочитай `claude-memory/bases/*.md` — загрузи список баз
4. Прочитай сессии за сегодня (или последний день с сессиями) — пойми текущее состояние
5. Спроси пользователя: "К какой базе относится? 1) {список баз} N) Новая база"
6. Создай НОВЫЙ файл сессии: `claude-memory/sessions/{YYYY-MM-DD HH-mm} — {Описание}.md`
   - Каждый диалог = НОВАЯ сессия (даже если за сегодня уже есть другие)
   - bases: ["[[выбранные базы]]"]
   - continues: "[[последняя сессия в базе]]" (если продолжение)
   - status: in_progress
   - Описание — краткое, по теме первого запроса

### ЖЁСТКОЕ ПРАВИЛО: Запись изменений файлов

**После КАЖДОГО изменения файла** (Edit, Write) — СРАЗУ дописать в сессию `## Затронутые файлы`:

```
- [[путь/к/файлу]] — что именно изменилось
```

Это НЕ опционально. Это НЕ "потом запишу". СРАЗУ после каждого Edit/Write.
Без wikilinks память отключена от проекта. Без описания — бесполезна.

Один диалог = одна сессия. Статус `in_progress` до конца диалога. НИКОГДА не закрывать сессию досрочно.

### В процессе работы:
- Записывай ключевые решения в файл сессии
- Все memory-операции — через `claude-memory/`, НЕ через `~/.claude/projects/.../memory/`

### Завершение сессии:
- Обнови статус сессии: `status: completed`
- Добавь `## Следующие шаги`
- Обнови `claude-memory/MEMORY.md` индекс

## Языки

- **Публичные файлы проекта** (скилл `~/.claude/skills/obsidian-memory-skill/*.md`, README.md, код, комментарии) — ТОЛЬКО английский
- **claude-memory/** (сессии, память, базы, MEMORY.md) — русский
- **CLAUDE.md** — русский (локальный файл проекта)

При изменении скилла или документации — пиши на английском. Русский текст в публичных файлах = баг.

## Два уровня правок

После тестирования изменений на локальной сборке:
1. Обнови **исходники** проекта (`obsidian-claude-memory/src/`, скилл в `~/.claude/skills/`)
2. Обнови **публичные файлы** в репо (`skills/obsidian-memory/` если есть, README.md)
3. Пересобери и задеплой плагин
4. НЕ пушить/релизить пока оба уровня не синхронизированы
